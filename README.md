# AI & Green Skills – Data Analytics and Machine Learning Tasks

This repository contains tasks and experiments completed as part of an **industry-backed internship under the Skills4Future Program**, focused on **Artificial Intelligence, Data Analytics, and Green Skills**.  
The work primarily involves **data preprocessing, exploratory data analysis, and implementation of machine learning algorithms** using real-world datasets related to energy consumption, environmental factors, and customer behavior.


---

## Objectives :

- Understand and apply **data cleaning and preprocessing techniques**
- Perform **exploratory data analysis (EDA)**
- Implement **supervised and unsupervised machine learning algorithms**
- Analyze **energy consumption and environmental datasets**
- Gain practical exposure to **AI applications for sustainable and green technologies**

---

## Implemented Concepts & Algorithms :

- Simple Linear Regression
- Logistic Regression
- K-Means Clustering
- Data Handling using Pandas
- Exploratory Data Analysis (EDA)
- Feed Forward Neural Networks (FNN)
- Energy Consumption Prediction

---

## Datasets Description :

- **Mall_Customers.csv** – Customer segmentation dataset for clustering analysis  
- **appliance_energy.csv** – Household appliance energy consumption data  
- **environmental factors.csv** – Environmental parameters affecting energy usage  
- **green_tech_data.csv** – Green technology-related indicators  
- **predict_energy_consumption.xls** – Dataset for energy consumption prediction
